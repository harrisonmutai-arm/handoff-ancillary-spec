Introduction
============

This is an auxiliary document of the Firmware Handoff Specification [FWH]_.
This document contains the definitions of Transfer Entries (TE) that are specific to a project or codebase.
The main Firmware Handoff Specification [FWH]_ contains TEs that are considered generic with wide applicability in different firmware codebases.
